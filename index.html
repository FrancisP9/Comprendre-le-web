L'Évolution du Web

1989 - La création du World Wide Web

Le Web a été inventé en 1989 par Tim Berners-Lee au CERN. Il proposait un système de partage d'informations basé sur l'hypertexte, permettant de naviguer entre des documents reliés entre eux.

1991 - Première page Web en ligne

En 1991, la première page web est mise en ligne. C’était une simple page informative sur le projet World Wide Web.

1993 - Premier navigateur graphique

Mosaic, le premier navigateur graphique, voit le jour en 1993, rendant la navigation web plus visuelle et accessible au grand public.

2000s - L’avènement du Web 2.0

Le début des années 2000 marque l’ère du Web 2.0, centré sur l'interaction des utilisateurs, la création de contenus dynamiques et les réseaux sociaux.

2010s - Web mobile et applications

Avec l’essor des smartphones, le web devient plus mobile, avec des applications et des sites optimisés pour les appareils mobiles.

2020s - L'ère du Web 3.0

Le Web 3.0 marque une nouvelle évolution avec la décentralisation, les technologies blockchain, et l’intelligence artificielle pour rendre le web plus intelligent et sécurisé.

Web 1.0

Le Web 1.0, souvent appelé le "Web statique", est apparu dans les années 1990. Il était principalement composé de pages HTML statiques où les utilisateurs consommaient l'information sans interagir directement.

Web 2.0

Le Web 2.0, apparu dans les années 2000, a transformé le web en un espace interactif. Les utilisateurs pouvaient publier, commenter et interagir via des plateformes comme les réseaux sociaux et les blogs.

Web 3.0

Le Web 3.0 est la prochaine génération d'internet, intégrant des technologies telles que la blockchain, la décentralisation des données et l’intelligence artificielle pour créer un web plus sécurisé et personnalisé.

Normes du W3C : Garantir l'Interopérabilité du Web

_ Normes W3C en cours...
_ Accessibilité : Aider tout utilisateur à accéder au Web
_ Compatibilité : Assurer que le contenu fonctionne sur tous les navigateurs
_ Sécurité : Garantir des communications Web sécurisées
_ HTML, CSS, JavaScript : Fondations essentielles des normes du Web
_ Bonne pratique : Conformité avec les standards mondiaux

Les normes W3C sont la pierre angulaire du développement web moderne. Elles assurent l’interopérabilité entre les navigateurs et plateformes, garantissant un web ouvert et accessible à tous. Les développeurs doivent respecter ces standards pour créer des expériences cohérentes, sécurisées et inclusives.

Sécurité Informatique : Virus, Phishing et Hacking

Virus

Un logiciel malveillant qui se propage d'un fichier à un autre pour endommager un système ou voler des données personnelles.

Phishing

Une méthode d'escroquerie qui consiste à tromper les utilisateurs pour qu'ils divulguent des informations sensibles via de faux e-mails ou sites web.

Hacking

Le piratage est l'accès non autorisé à un système informatique, souvent pour voler ou corrompre des données.

Grabbing

Une technique consistant à capturer des identifiants de connexion ou des informations confidentielles en interceptant des données.

À l'ère numérique, la sécurité informatique est cruciale. Les cyberattaques, comme le phishing, le grabbing, ou les virus, sont en constante augmentation. Connaître ces menaces permet de mieux se protéger.

Technologies : NoSQL, Node.js, Markup, AJAX

NoSQL

NoSQL est une approche des bases de données non relationnelles, conçue pour stocker et gérer de grands volumes de données non structurées ou semi-structurées.

Node.js

Node.js est une plateforme JavaScript côté serveur, permettant de créer des applications réseau rapides, évolutives et non bloquantes.

Markup

Le langage de balisage (Markup) désigne les langages comme HTML ou XML qui sont utilisés pour structurer et organiser des contenus textuels.

AJAX

AJAX (Asynchronous JavaScript and XML) permet d'actualiser les pages Web de façon asynchrone sans les recharger complètement.

Métiers du Web

Développeur Web

Le développeur web conçoit et crée des sites et applications web interactifs en utilisant des langages comme HTML, CSS, et JavaScript.

Designer UX/UI

Le designer UX/UI s'assure que l'interface d'un site web est intuitive et centrée sur l'utilisateur, tout en étant esthétiquement agréable.

Expert SEO

L'expert SEO optimise les sites web pour les moteurs de recherche afin d'améliorer la visibilité et le classement dans les résultats de recherche.

Analyste en Sécurité

L'analyste en sécurité protège les systèmes d'information contre les cyberattaques. Il analyse les vulnérabilités, met en place des solutions de sécurité et surveille les incidents.

Focus : Le Métier d'Analyste en Sécurité

Un analyste en sécurité informatique est chargé de protéger les systèmes d'une organisation contre les cybermenaces. Son travail inclut l'identification des failles de sécurité, la mise en place de mesures de protection, la surveillance des réseaux, et la réponse aux incidents de sécurité.

Compétences requises :

  • Connaissances approfondies des menaces numériques (malware, phishing, etc.)
  • Maîtrise des outils de surveillance et de détection d'intrusion
  • Gestion des incidents de sécurité
  • Capacité à élaborer des stratégies de sécurité

Conclusion

Le web continue d’évoluer rapidement, avec de nouvelles technologies, des métiers en expansion, et des menaces à surveiller. Une bonne compréhension de ses principes fondamentaux est cruciale pour naviguer en toute sécurité et exploiter son plein potentiel.
